IrDA Tools

SUMMARY

The \tools\irda folder includes three tools used in developing IrDA miniport drivers and testing IrDA adapters.

·         DbgCon: Debugs IrDA miniport drivers. Use in addition to or instead of the kernel debugger.

·         Sledge: Administers a stress test for IrDA miniport drivers and the IrDA protocol driver, irda.sys.

·         Sniffir: Monitors infrared packets transmitted between two computers.

Help files for these tools are included in the Windows DDK Documentation in the "Tools for Developing IrDA Miniport Drivers" topic (Windows DDK, Network Devices and Protocols, Design Guide, Miniport Drivers, IrDA Miniport Drivers, Tools for Developing IrDA Miniport Drivers).
