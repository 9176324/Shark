Stdunk

SUMMARY

The Stdunk sample is the standard implementation of the CUnknown class and the IUnknown interface. The Stdunk sample code is required to build the WDM audio sample drivers. It is not included as an example to be modifed by developers, as is the case with the other DDK samples, but simply as code that is required to build any WDM audio driver.
