4mm DAT Tape Minidriver

SUMMARY

This sample is a working example of a 4mm DAT Tape Minidriver. This module contains device-specific routines for 4mm DAT drives such as Sony SDT-2000, Sony SDT-4000, DEC TLZ06, Seagate, Hewlett-Packard 35470A, Hewlett-Packard C1533A, and so on.

BUILDING THE SAMPLE

To build the sample, run the build command. The lib file Tape.lib is required to build the sample.

Before you can use the sample, you need to install a tape drive that uses 4mmdat.sys. The list of drives that use 4mm DAT is given in Tape.inf in the %SystemRoot%\Inf directory. This file is used by the system to install. If you make any changes to Tape.inf, be sure to delete Tape.pnf, if it exists.

To install the drive, shut down the machine, attach the drive, and boot. The drive will be detected and Microsoft ® Windows ® Server 2003 will prompt for the 4mmdat.sys driver. The driver will be copied into %SystemRoot%\System32\drivers.

Building this sample produces 4mmdat.sys. Both free and checked versions are buildable. The sample has been tested with a binary that was built with Microsoft Visual® C++ 6.0.
